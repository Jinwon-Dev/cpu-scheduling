# cpu-scheduling

Java를 이용해 CPU 스케줄링 알고리즘을 구현해 봅니다.

- **선입 선처리 스케줄링(FCFS : First-Come, First-Served Scheduling)**
- **최단 작업 우선 스케줄링(SJF : Shortest-Job-First Scheduling)**
- **라운드 로빈 스케줄링(RR : Round-Robin Scheduling)**
- **우선순위 스케줄링(Priority Scheduling)**
- **다단계 큐 스케줄링(Multilevel Queue Scheduling)**

</br>

CPU 스케줄링에 대해 자세히 정리한 학습 내용은 [개인 노션](https://jinwonyoon.notion.site/Chapter-5-CPU-f5f1cbc6867c4267b606fa5199f87fac)과 [TIL 레퍼지토리](https://github.com/Jinwon-Dev/TIL/blob/main/OS/operating-system-concepts/cpu-scheduling.md)에서 볼 수 있습니다.
